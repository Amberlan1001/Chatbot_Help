一、Chatbot_Help介绍
==========================

    Chatbot_Help是一个接入机器人的第三方扩展工具，提供一系列相关api来将自己开发的机器人（Chatbot）接入到相关外部平台，实现与用户的交互

目前，Chatbot_Help支持的第三方平台有：

1、钉钉群：

2、微信公众号；

3、QQ群；

4、外部平台：如电商平台、网页端等


二、安装使用
============

::

    pip install Chatbot_Help
